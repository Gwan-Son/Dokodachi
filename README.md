# Dokodachi
 
### 중간 진행사항 점검
Socket.IO를 통해 실시간 채팅 기능 구현.
상대방 메시지와 내 메시지를 구별하여 leading, trailing을 각각 설정함.
상대방 메시지에는 username이 보일 수 있게 표시.
메시지를 보낸 시간은 둘 다 표시.
